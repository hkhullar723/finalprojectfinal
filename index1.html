Css cheat sheet


Css tands fro cascading style sheet which is used to add colours, designs to a webpage. Some useful css code snippets, tags and attributes are as follows:

1. Background colour: this tag is used to add color to a background for a HTML element.
2. Padding: this tag is used to add padding inside a html element tag from its sides. It gives some space around the text along the edges of a text or box.
3. Colour: this tag allows user to change colour of the text as their wills.
4. display tag- It is used to cha ge to diplay as of respective div or a paragraph. /* Changing display */ .myBlock { display: block; } /* Changing position */ .positionElement { position: relative; top: 20px; left: 10px; }
5. border properties- it is used to set width, height and colour of the border around a text.
6. font- using font selections, you can change the font. numerous number of fonts are available online which u cam download and add to your css styling.

/* Font size */
font-size: 16px;
/* Font family */br font-family: 'Arial', sans-serif;
/* Font weight */
font-weight: bold;
/* Text alignment */
text-align: center;

7. background image - this tag is used to insert a background image in the webpage.
8. background rgb- this tag is used to set the opacity or transparency to their own will so in order to display the content on the background image.
9.transition tag is used to set the transition for a particular element.
10. grid - a grid layout can be set, fixing the shape and columns for it using css tags.
.grid-container {
display: grid;
grid-template-columns: repeat(3, 1fr); /* Three columns with equal width */
grid-gap: 10px; /* Gap between grid items */
}
